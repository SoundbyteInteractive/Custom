Built for your stream,
not the masses.

This isn't a plugin you download and forget. Every build is personally crafted for a single streamer. The character, the events, the humor, the chaos, all calibrated to your community. I am only personally reaching out to a limited number of streamers to keep each build as unique as possible.

What software does it work with?

OBS Studio and Streamlabs OBS. Added as a browser source, no plugins required.

Do I need coding experience?

None at all. The setup guide walks through every step. If you can add a browser source in OBS, you can run this.

Will anyone else have the same one?

No. Every build is exclusive to the streamer it's made for. Your character and events will never appear on another channel.

How do I get started?

Commission via the button above, or reach out directly at Brightsidebookings@gmail.com. We'll set up a quick call or chat to go over your stream and vision.
